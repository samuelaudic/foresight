# Foresight API

Backend API for a personal finance application.

The goal of this project is to experiment with a clean, modular Spring Boot architecture while building a small but realistic finance-oriented backend.

---

## Tech stack

- Java 17
- Spring Boot
- PostgreSQL
- Flyway
- Maven

---

## Commit Convention

This project follows a simplified Conventional Commits format:

<type>: <short description>

Allowed types:
- feat
- fix
- chore
- refactor
- docs
- test

---

Early MVP – work in progress.
